# Ironclad Inspections — ironcladinspections.ca

## What This Is
A business website for Ironclad Inspections, a home inspection company in Southern Alberta.
Built and managed by Carol at DayOnePros (dayonepros.com) under a monthly retainer.

## Hosting & Deployment
- Hosted on GitHub Pages
- Will be connected to custom domain: ironcladinspections.ca
- Currently under builtbycarol GitHub account, will transfer to Mike's account later

## Tech Stack
- Plain HTML, CSS, and vanilla JavaScript only
- No React, no Node.js, no npm, no build tools
- All files must work when served as static files on GitHub Pages
- Contact form uses an external service (not server-side processing)

## Project Structure
- /index.html — main homepage
- /css/ — all stylesheets
- /images/ — all images and graphics
- /js/ — any JavaScript files

## Rules
- NEVER delete or overwrite index.html without asking me first
- NEVER add frameworks, libraries, or package managers
- NEVER change the folder structure without asking me first
- Keep all CSS in /css/, all images in /images/, all JS in /js/
- Every change should be small and testable in a browser
- All new pages must be mobile-responsive
- Use semantic HTML (header, nav, main, section, footer)
